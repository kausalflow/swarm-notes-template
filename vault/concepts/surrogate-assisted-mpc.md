# Surrogate-Assisted Model Predictive Control (MPC)

> *Auto-generated stub. Edit this file to add more details.*

A control framework where a learned operator (surrogate model) is used iteratively within a Model Predictive Control loop to solve for optimal control actions.


## Why It Matters

This framework directly addresses the computational bottleneck of using high-fidelity numerical models for real-time control by replacing them with fast surrogate models (learned operators).
